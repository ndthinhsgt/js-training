Mảng dữ liệu & Object

Nội dung

        - Mảng: tasks là chứa các tasks
        - Mảng: doneTasks là chứa các tasks đã được đánh là done
        - Mỗi phần tử của mảng là 1 object có key là: label: chuỗi, description: chuỗi
    
        

        Mô tả hoạt động:
        - Khi thêm task thì label, description lấy từ form và được push vào tasks
        - Khi done task thì task này bị xoá khõi tasks, và được push vào doneTasks
